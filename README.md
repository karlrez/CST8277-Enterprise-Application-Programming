# CST8277-Enterprise-Application-Programming

Assignments and labs related to Java EE. Each Assignment also contains a pdf with more information regarding the specific project. 

## Assignment 1
- Creating a Managed Bean to manipulate the model class
- Creating a DAO class to manipulate the database
- Creating a JSF application that supports CRUD operations

## Assignment 2
- Using JSF in conjuction with EJBs and JPA
- Creating a model class for JPA
- Adding validation to JSF views

## Assignment 3
- Mapping our database tables and JPA entitites
- Building JPQL queries
- Includes JUnit test cases 

## Assignment 4
- Putting all the previous knowledge together into a REST API project
- Setting up JEE Security roles
